MRHB’s Halal Staking Solution MIRO Aces Hacken Audit

-

2022-10-02 • 1 min read

Table of content

  • About M.I.R.O.
  • About MRHB.Network

We are proud to announce that our soon-to-be-launched Marhaba Incentivization and Reward Offering (M.I.R.O.) halal staking platform has passed Hacken’s Smart Contract Code Review and Security Analysis with an overall score of 9.8/10!

Hacken is a leading blockchain cybersecurity consulting company that has also previously audited MRHB DeFi BEP-20 token that passed with flying colors.

The score for M.I.R.O. security audit was based on the following components:

  1. Documentation quality (10/10)
  2. Code quality (10/10)
  3. Architecture quality (8/10)
  4. Security score (10/10)

M.I.R.O. scored perfect scores with respect to documentation quality, code quality and also its security score.

With regards to the architecture quality component, the development environment and the description of how to compile, build and test the code were provided. MRHB will endeavor to improve on its architecture quality.

A more detailed description of the report can be found here.

 

About M.I.R.O.

M.I.R.O. is a halal staking platform that gives rewards and voting power to users who stake $MRHB. The longer a user stakes $MRHB, the higher the share of reward. M.I.R.O. is planned to be launched within Q4 of 2022.

Be sure to get connected on our socials (linked below) so you don’t miss out on the launch of M.I.R.O. and other halal DeFi solutions within the MRHB Ecosystem.

About MRHB.Network
